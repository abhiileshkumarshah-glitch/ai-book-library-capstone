# AI Book Library Assistant

This is a full-stack AI-powered web application built for the capstone project.

## Tech Stack

- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL
- AI: Gemini API
- Deployment: Docker Compose

## Features

- Add, view, update, and delete books
- Track reading status and rating
- Add reading notes
- AI book recommendations using saved book data
- AI note summaries
- AI agent that can update book status and rating

```bash
docker compose up --build
